## Install

```sh
yarn install
```

```sh
npm install
```

## Usage

```sh
yarn run start
```

```sh
npm start
```

## Env vars

| NAME              |                                                            |
| ----------------- | ---------------------------------------------------------- |
| REACT_APP_API_KEY | your [Kutt](https://kutt.it/) api key (used for demo mode) |
